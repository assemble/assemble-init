# assemble-init [![NPM version](https://badge.fury.io/js/assemble-init.svg)](http://badge.fury.io/js/assemble-init)

> Plugin for initializing an assemble pipeline.

## Install
## Install with [npm](npmjs.org)

```bash
npm i assemble-init --save
```

## Run tests

```bash

```

